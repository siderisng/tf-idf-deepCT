<ABSTRACT>
in this converter, the microcomputer is supplied with the analog variables to be multiplied via analog/digital converters (2).  if it is desired to detect the analog variables with an accuracy of 1% over a relatively large measuring range, an a/d converter (2) and a microcomputer (1) having a corresponding resolution are necessary.  to be able to manage with microcomputers (1) and, particularly a/d converters (2) which are not so elaborate with the corresponding measuring range, an amplifier (3), which is supplied with the analog variables via resistors (6-11) of a resistor network (4), is connected in series preceding the a/d converter (2) in accordance with the invention, in which arrangement the appropriate resistor values can be preselected by the microcomputer (1).  <image>
