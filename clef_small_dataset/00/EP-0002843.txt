<TITLE>
windscreen wiper for a motor vehicle.
</TITLE>
<APPLICANT>

<ABSTRACT>
the device comprises a wiper blade having a pressure distribution bracket (14) which is part of a carrier frame and has two lateral cheeks (18, 20) arranged at a distance from one another.  mounted in the cheeks (18, 20) is a hinge bolt (22) which projects laterally out of the pressure distribution bracket (14).  a connecting element (28) is mounted in articulated fashion on this hinge bolt (22), to which connecting element a wiper arm (10) can be connected.  the connecting element (28) is connected via a web (64) to an auxiliary element (58) which is mounted so as to be capable of oscillating between the lateral cheeks (18, 20) of the pressure distribution bracket (14) on the hinge bolt portion located there.  the web (64) extends from the connecting element (28) via the one cheek (20) to the auxiliary element (58).  <image>
