<ABSTRACT>
a variable level converter comprises a first variable level  converting unit (110), a second variable level converting unit  (120) and a grounded-emitter amplifying transistor (13) con­ nected between them.  each of the first and second variable  level converting units is comprised of an inverted l-type var­ iable attenuator which includes a series resistor (111,121)  and a variable impedance element (112,122).  the second var­ iable level converting unit includes the collector resistor (13)  of the transistor circuit.  each variable impedance element is  controlled by the same level control signal (sc) and has the  same electric characteristics.
