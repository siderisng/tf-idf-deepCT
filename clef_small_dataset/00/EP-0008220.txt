<ABSTRACT>
an interface between a telephone switch bus line (14)  and a dual tone multiple frequency (dtmf) type digital  tone decoder (12) is provided which minimizes false tone  decoding due to filter-induced transient oscillations (44)  caused by spurious impulse noise (40) and the like on the  input bus line (14).  the interface includes a limiter circuit  (20, 22) for generating a uniform amplitude output pulse  train (50) and a circuit (24) for automatically generating a  variable threshold which is time responsive to input signals  on the input bus line (14).  the variable threshold is used  by the limiter circuit (20, 22) to mask transient oscillations  (44) in the tone filter output.  only when persistent oscillaÂ­ tions having a peak amplitude less than the threshold level  are delivered to the limiter from a tone filter (16, 18) will  input signals be recognised as valid switching tone signals.
