<ABSTRACT>
Methods and apparatus for taking into account the peculiarities of the various types of films in color copying, wherein values for the copying properties and possible mask densities are input to the exposure control device for each film type in storage, which are switched in accordance with the film variety.To simplify the submission to different types of films, the information about the respective movie variety is entered as machine-readable code in the film (28) and this code is read in a copier and the memory switching is automatically made.
