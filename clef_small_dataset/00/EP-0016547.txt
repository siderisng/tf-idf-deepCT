<TITLE>
fuel control system for an internal combustion engine.
</TITLE>

<ABSTRACT>
an i. c.  engine fuel control includes a main fuel control (19)  which control fuel flow to the engine in accordance with one  or more engine operating parameters.  a roughness sensor  (11) provides an electrical signal varying with the magnitude  of fluctuations in the engine speed and an integrator (15)  integrates the error between this signal and a reference  signal.  two active clamp circuits (21,31) operate to limit the  excursion of the integrator output which provides a limited  authority trim input to the fuel control.
