<ABSTRACT>
The invention relates to a storage program Miable control with a control unit, a per gram memory, input module and output assembly PEN.According to the invention, a memory for receiving a process image is provided, which is renewed in each machining cyclus.The process binder is an image of the signal states of the inputs and the output states of the outputs.During the processing of a control process, the control unit does not work with the input and output modules, but exclusively with the per zeta image.As a result, the processing speed is increased and a fluttering of the outputs is prevented.
