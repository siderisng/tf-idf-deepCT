<ABSTRACT>
Data input / output controller between the main pro cessor and the input / output units in a data processing system. The controller (10) COM takes a first register (30) to send data to the main processor (20) and to receive data from this processor, a second register (32) for sending data to the input units / Output (13-16) and to receive data from these units, a high speed data transfer bus (31) between the two registers, and the automatic data transfer control circuits (35). The mid croprocessor (33) supervises the synchronization of opera tions in the controller and provides the initialization of the automatic control circuits prior to the beginning of the transfer operations. Such an input / output controller allows the automatic transfer of the data between the main pro cessor and an input / output unit without intervention of the microprocessor.
