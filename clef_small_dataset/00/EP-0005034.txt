<ABSTRACT>
an electronic image processing system for converting  character font data into image data utilizable by a raster  output scanning device, such as a ros printer, capable of  formatting the image data in a plurality of scan lines onto an  imaging medium includes a main memory (16) having  character font data stored therein and an input memory (110)  for receiving character font data from the main memory.  in order to achieve reproduction of characters of different  shades and tones, the system further includes an "ink" data  memory (106) responsive to a predetermined address signal  for providing a predetermined pattern of ink data defining a  desired grey-scale tone for the image data; a band buffer  memory (memory a or memory b) capable of  storing image data representing a predetermined number of  scan lines constituting an image band, the band buffer  memory having enable input lines and associated data input  lines; means (108) coupled to the input memory for addressing  the input memory to access character font data  therefrom; means (112, 114, 104, 116, 118, 100/184) coupled  to the band buffer memory for applying character font data  accessed from the input memory to respective enable input  lines of the band buffer memory; means (102) coupled to the  ink data memory for applying the predetermined address  signal to the ink data memory; means (100/186) coupled to  the band buffer memory for applying the predetermined  pattern of ink data accessed from the ink memory to  respective data input lines of the band buffer memory; and  means (102, 100/182) for addressing the band buffer memory  to load into an addressed memory location the predetermined  pattern of ink data only with respect to those data  lines whose associated enable input lines are enabled by the  character font data.  
</ABSTRACT>
