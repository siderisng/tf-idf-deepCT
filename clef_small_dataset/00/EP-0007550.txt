<ABSTRACT>
The invention relates to a rotary field drive with two identical alternating voltage integrators (IGR, Igs) to form an electrical voltage signal (ψr, ψS) at the respective integrator output, which is each proportional to a flow component in the machine (M). Each alternating voltage integrator (IGR, Igs) has an integrator (V1) and a zero point controller (F1, V2, F2, V3, V4) arranged in the counter-coupling branch for suppressing the equilibrium of the input voltages supplied to a sum point (S1) of the integrator (V1). Each integrator (V1) is supplied as input voltages in each case a stern voltage associated with the flow component proportional voltage (GE) and for compensating the ohmic and inductive stator voltage drop, a voltage proportional to the associated stand stream in a machine supply line. By reducing the phase errors of the generated flow components, optimum converter and machine utilization is ensured in an enlarged speed control range. Each zero point controller has a P controller (V3) and an I-controller (V4). The output signal of both controllers (V3, V4) is supplied to the sum point (S1). The output signal of the integrator (V1) corrected about the ohmic strip voltage drop and the inductive stray voltage is weighted proportional to the frequency of the machine (M) (A; F1) weighted to the input of the P controller (V3) and proportional to the square of the frequency (A; F1, F2) supplied to the input of the I controller (V4) with the weighting factor (A) comprises a maximum value of 1.
