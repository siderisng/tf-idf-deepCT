<ABSTRACT>
A hazard alarm system increases the availability of the system by decentralizing system intelligence.The entire system is divided into small, fully functional system blocks (SYB), with each system block (SYB) switching devices (E1, E2.,, EN) for one or more detectors (M) or for one or more subordinate system blocks(SYB), a programmed control (PST) has a query and evaluation of the detector signals as well as connections for display, operating and registrar.One of the system blocks (SYB1) serves as a central (Z), while the other system blocks (SYB 2, SYB 3, SYB 4) of this control panel in one or more hierarchical levels (K1, K2) are subordinate.
