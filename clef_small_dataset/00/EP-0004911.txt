<ABSTRACT>
Danger alarm system, for example fire alarm system, with a plurality of detectors (M11.,, mmn), which are connected via reporting lines (L1.,, LM) to a central (Z).In the central (Z) a memory (SP) is provided in which characteristic data is stored for each cleavable detector.From the individual detectors (M11.,, MMN), cyclically measured values (M) are queried, as well as set values (SWA) are formed from the stored data for each detector and compared with the measured values;From the comparison, noise or alarm signals are optionally derived.
</ABSTRACT>
