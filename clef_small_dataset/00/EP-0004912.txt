<ABSTRACT>
Hazard alarm system, for example fire alarm system, with a plurality of detectors (M1.,, Mn), which are connected via reporting lines (L1.,, Ln) to a central (Z).In the central (Z) a write-read memory (SP) is provided, in which for each of the reporting lines (L1.,, Ln) switchable detector (M1.,, Mn) is provided a memory location (SP1, SPN)on which the actual occupancy of the relevant detector's place is stored.For each query of the reporting lines (L1., LN), the reporting signals (A, S) coming from certain detector locations are evaluated only if a assignment is stored for the relevant detector locations.
