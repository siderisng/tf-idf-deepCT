<ABSTRACT>
circuitry for sequentially producing the function values (w) of several oscillations, the subsequent frequencies of which amount to n-times a fundamental frequency (sch1).  a counter (z) periodically outputs p counts (a) in binary representation.  a generator (ng) generates binary signals (n) which signal the numbers (n).  a multiplier (mu) multiplies the counts (a) in modulo p by the numbers n and the resultant multiplicative signals (an) are used for addressing a read-only memory (rom) which outputs the corresponding function values (w).  <image>
</ABSTRACT>
