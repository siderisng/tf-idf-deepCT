<ABSTRACT>
The invention relates to a programmable rear derailleur for controlling the timing of functions in devices. The derailleur arranges input signals (E) using a switching network (Sn) formed from at least one programming ba ren matrix, a state register (Zr) and at least one timer ZG1, ZG2) output signals (A). The switching net (Sn) is preceded by an input matrix (EM) which generates control signals (P) supplied from the input signals (e) in response to different priority levels (P) to the switching net (S). The priority signals (P) generated in a priority generator (Pg) are supplied to both the input matrix (EM) and the switching net (Sn), whereby a processing of the input signals (E) is achieved according to a time division multiplex operation. The switching net (Sn) is preferably formed from a main matrix (Hm) and an output matrix (AM).
