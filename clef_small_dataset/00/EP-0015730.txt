in the system carrier is continuously transmitted from  transmitting side (1, 2) to receiving side (4, 5).  outside effective data sending periods continuous "1"  bits (all mark, waveform [c] fig.  2) are sent.  to detect effective data sending periods, a code convert­ er (22) on the transmitting side (1, 2) inverts selected data bits  for sending.  the bit coinciding with a send request signal rising to "1"  is inverted (waveforms [b] and [d]; fig.  2), and subsequently  any bit which follows four consecutive bits of equal value is in­ verted whilst the send request signal is "1".  a receiving side (4, 5) detector (45) detects the first in­ verted bit (first "0" bit) and then detects subsequent inver­ sions.  the beginning and continuation of an effective sending  period are thereby determined.  when no inversions are detected for a predetermined  number of received bits (e. g.  48) it is determined that the peri­ od has ended.  a decoder (44) restores the original values to detected in­ vertical bits.
