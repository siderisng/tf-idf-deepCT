<ABSTRACT>
the invention relates to an electronic programmer in which switching command words and time interval command words are digitally stored in two memories.  the switching command words determine the switching state of switching elements and the time interval command words determine the duration of such switching states.  an address generator generates for each switching command word and for each associated time interval command word an address which simultaneously addresses the two last-mentioned words.  a timer can be preset and started correspondingly by each addressed time interval command word.  after a time interval given by the time interval command word has elapsed, the address generator is caused to access with the next address.  an existing interface also converts each switching command word addressed into compatible signals for the switching elements.  the programmer has short switching times, exhibits a simple configuration and operates extremely reliably.  <image>
