<ABSTRACT>
in a small-sized box with two or three folding sides there are disposed alphanumeric data input means in the form of a typewriter keyboard, means for storing the data introduced which comprise a permanent scratch-pad memory, a text memory which can contain a large number of characters and a line memory associated with a row of alphanumeric display devices.  the unit operates through control means which permit the writing, erasing, modification, searching and displaying of data or data blocks in the memories, and word processing facilities.  <image>
