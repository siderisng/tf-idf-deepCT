<ABSTRACT>
the network comprises a random-access memory (m), k arithmetic units (u0 to uk-1) disposed in parallel, each interacting with a read- only memory (h0 to hk-1) storing the weighting coefficients.  the samples to be interpolated are written into the read-only memory on each occasion instead of the oldest sample, the delayed discrete values being read commencing with the sample just written, and progressing in increasing order of age.  application: time-share processing of signals in pcm telephony.  <image>
</ABSTRACT>
