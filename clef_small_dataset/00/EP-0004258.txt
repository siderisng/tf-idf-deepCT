<ABSTRACT>
The invention relates to a computer tomograph in which the tomography processor contains a corrector (112) for correcting the output signals of the detector (14).The corrector (112) is followed by a first memory (120) for storing the corrected data of a projection.This includes a convolution calculator (114) for the folding of the corrected data of a projection.This is followed by a second memory (122) for the folded data points, which controls an image memory (118) for storing data representative of the density of the body in the cross-sectional plane.Further, a recontructor (116) is present, which causes the folded data points of a projection to be reproduced by a corresponding assignment for each place in the image memory (118) by adding the value in this space in the image memory (118) after appropriate weighting.
