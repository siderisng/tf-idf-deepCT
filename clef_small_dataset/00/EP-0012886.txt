<ABSTRACT>
Input / output control unit for transferacting data between a main processor (12) and plus input / output devices (14) in a data processing system.When the control unit (10) receives service requests from the input / output devices (14) it transmits an interrupt signal to the main processor (12) via the interuption logic (56).The latter then generates a transfer command that allows the control unit to transfer service requests to the processor.The interrupt logic (56) is decommissioned the transfer control so as to prevent the interrupt signal from the main processor from being generated while service requests are pending in the control unit.
