<ABSTRACT>
The present invention relates to a device for a television titration device.The device allows the device to present characters of a legible language from the right to the left.For this purpose the device comprises: a main memory (311) for storing features representations corresponding to symmetric or inverse images of a set of characters of the language considered;an auxiliary memory (520) for storing a line bits analysis line from the feature generator (300) of the device;and means (530, 550) for extracting the auxiliary memory the line bits analysis line in reverse order of that where the line bits were introduced into the auxiliary memory.application to the field of television.
