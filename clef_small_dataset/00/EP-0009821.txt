<ABSTRACT>
in this system, the code carrier (6) indicates the position of the weight switching mechanism in an analog manner and exhibits, in dependence on the possible switch positions, several adjacently located code tracks (s 1 to s 5) which are subdivided into magneto-electronically and/or opto-electronically active and inactive sections (7, 8) and correspond to a sensing arrangement exhibiting several transmitter/receiver units (1, 1' to 5, 5') which is connected to an indicating and/or control device.  to solve the problems of interfering superimposition of the transmitter/receiver signals of adjacent code tracks in the miniaturisation of the sensing arrangements, the transmitter/receiver units (1, 1' to 5, 5') allocated to the individual code tracks (s 1 to s 5) are arranged offset with respect to one another in the longitudinal direction of the code tracks (s 1 to s 5) and the magneto-electronically and/or optoelectronically active and inactive sections (7, 8) of the respective code tracks are offset with respect to one another in corresponding manner.  <image>
