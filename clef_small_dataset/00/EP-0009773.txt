<ABSTRACT>
the circuit contains a phase comparison circuit (2) which receives the input signal at a first input (a) and outputs a phase difference signal at an output (c), which signal continuously contains the phase difference between the input signal and a comparison signal which is supplied via a feedback branch (4, 6) from the output (c) to the second input (b) of the phase comparison circuit (2) at a frequency which tracks the instantaneous frequency of the input signal at a predetermined rate of change.  the phase comparison circuit (2) is followed by an output circuit (14) which with a constant phase difference within a predetermined time interval indicates the presence of the tone signal and with a variable phase difference within the predetermined time interval indicates the presence of the speech signal.  the invention also relates to a method in accordance with which the circuit operates.  <image>
