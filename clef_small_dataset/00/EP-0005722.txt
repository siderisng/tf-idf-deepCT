<ABSTRACT>
System for selecting interface circuits placed between applicable priority service bodies such as adapters and a central unit such as a communication controller.The rank adapter "I" records in API the highest priority level among the requests for services formulated with it by the elements it serves.A comparison is carried out asynchronously between the API level and priority levels of Range I + 1 adapters to N.A Select SER line spreads downstream upstream the result of said comparison and also serves to engage one of the s flip-flops and restore the others.The actual selection of the adapter to be used will be limited to the location of the flip-flop.application to data communication systems.
