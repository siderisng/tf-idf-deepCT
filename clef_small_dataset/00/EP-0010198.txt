<ABSTRACT>
In a data processing system with in pages shared virtual memory, data pages between main memory and secondary memories are exchanged.The exchange happens due to the use of the pages.Frequently used pages are held in the main memory. In addition, input / output devices may require that certain pages are recorded.For this purpose, the address converter, which converts virtual addresses in main pond wheels, includes a page directory (24), which has a field for specifying the side use.Both the Zen Trelle Calculation Unit and the input / output units may cause this field to be described.
