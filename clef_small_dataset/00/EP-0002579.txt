<ABSTRACT>
the invention concerns a method of creating a data file  and a data processing network arranged to carry out such  method.  an embodiment of the invention provides a file security  system for data files associated with a host data processing  system.  the host system includes a data security device 11  which contains a secure host master key 13 and is capable of  performing a variety of cryptographic operations.  at initialization  time, the host system generates a series of secondary  file keys for the associated storage media and protects them  by enciphering the secondary file keys under a variant of the  host master key.  when a data file is to be created, a random  number is generated and defined as an operational key  enciphered under the secondary file key of a designated  storage media.  the host data security device, using the  enciphered secondary file key of the designated storage  media, transforms the enciphered operational key under  control of the host master key into a form which permits the  operational key to be used for enciphering host data.  the  operational key enciphered under the secondary file key of  the designated storage media, as header information,  together with the host data enciphered under the operational  key is written on the storage media as an enciphered data  file.  when the data file is recovered, the host data security  device, using the enciphered secondary file key of the  designated storage media, transforms the enciphered operational  key header information under control of the host  master key into a form which permits the operational key to  be used for deciphering the enciphered data file to obtain the  file data in clear form.  
</ABSTRACT>
