<ABSTRACT>
For a system for interactive cable television, the information distribution - for. b. per cable unrestricted 12 distribution programs or restricted 30 individual television programs - in the overall orientation area of ​​all utilities (I, II, III.,,,, Participant network level (E) is common in the star. In the uppermost network level (A) between the control panel (1) and the nodes (2), all utilities obtain the same distribution programs via a common cable (X). Individual programs run for each node (2) via own cables (Y) in the relevant supply area. At each node (2), a tree net (planes B, C, D) connects with parallel autonomous cables, which extends up to pre-field devices (3). The subscriber terminals (4) are connected to the apron devices (3) via simple cables (X). Rear ducts are guided via the distribution network. To increase the return channel capacity, an additional cable (Z) can be provided in the uppermost network plane (A).
