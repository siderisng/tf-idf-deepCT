<ABSTRACT>
a radio system having a central receiving station (figure 2) is proposed to which several subscriber stations connected via electrical lines are connected, and with at least one transmitting station (figure 1).  the transmitting station comprises a memory device (13) for storing subscriber addresses consisting of code words by means of a dialling switch arrangement or numerical key arrangement (10), the stored addresses in each case being transmitted by a switching command of the central receiving station.  after the code word allocated to the numerical key, a delay code word is stored in the memory (13) by means of a switching command, particularly by a prolonged operation of a numerical key.  the receiving station contains a detection device (33) for this delay code word by means of which the code words following the delay code word are delayed.  in a radio telephone system, this delay code word is preferably inserted after the sequence of preselection digits in order to allow the subscriber dialling device (34) sufficient time for finding a free tie line.  otherwise, there would be the risk that parts of the sequence of subscriber digits following the preselection sequence of digits would be lost.  <image>
