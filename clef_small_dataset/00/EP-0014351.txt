<ABSTRACT>
The invention is concerned with the task of developing a monolled theromatically integrable current driving circuit with high output voltage resistance, which is universally applicable for the power supply of the common logic circuits. For this purpose, according to the invention, a logic circuit with a NAND function and a Darlington stage serving as a signal outlet is beaten with the following construction before: the input transistors (Te₁, T₂.,.) The logic circuit form a differential amplifier with a common reference transistor (TR) DV) whose non-inverted exit is placed on the Darlington stage (DA) and the sen inverting output to a node (K₁). This node (K₁) forms the output of a transistors (T₁, T₂) and diodes (Z₁, Z₂ d₁, D₂) switching system (US). This switching system is designed so that the a passage transistors can be switched to different threshold voltage values.
