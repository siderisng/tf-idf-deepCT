<ABSTRACT>
The invention relates to a circuit arrangement for coding or decoding with a device for forming control bits using a linear code.The coding should be adapted to a desired coding mode as easy as possible.The invention provides for this purpose a binary memory (1), which provides a point of binary information a associated control point word.The memory (1) is followed by a modulo-2 adder circuit (2) for recharging control position words, depending on the binary information (K).An inventive circuit arrangement is particularly suitable for use in telecontrol equipment.
