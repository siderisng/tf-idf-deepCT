controller programmable with logic instructions representative of circuit nodes and elements of a ladder diagram.

<ABSTRACT>
a controller for electrical circuit systems and apparatus,  the controller being programmable with logic instructions  representative of circuit nodes of a ladder diagram.  as compared  to the prior art, this controller is simple to program and  does not require any modification of the conventional ladder  diagram.  the controller comprises a store wnm of the logic  values of the circuit nodes and a store ccm of the logic values  of the control coils in the ladder diagram.  the control program  is stored in main memory mm.  processing circuit lc1 is provided  for computing and updating the logic values of the  circuit nodes or control coils based on the previous stored  values and input signals from input circuit ib, and for storing  the updated logic values in the stores wnm and ccm.
