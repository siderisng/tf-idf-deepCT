<TITLE>
automatic dispensing device for lottery tickets
</TITLE>
<APPLICANT>

<ABSTRACT>
for the automatic issue of tokens, there is a device which has a vertical or inclined token shaft (2).  flat tokens (7) are stacked in this token shaft (2).  a suction pipe (5, 25, 26) is arranged above each token stack; for this purpose this suction pipe (5, 25, 26) is arranged vertically displaceably and pivotally on a stand (1).  each suction pipe (5, 25, 26) is connected to a vacuum-generating pump (41).  arranged laterally on or behind the token shaft (2) is an output funnel (12) having an oblique guide plate (11), the top edge of which extends in the region of the upper end of the token shaft (2).  <image>
