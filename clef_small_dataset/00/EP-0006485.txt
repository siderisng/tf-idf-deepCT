<ABSTRACT>
PAGE addressing device in a processing system with a data page addressing register (27) and a page addressing register of inputs (28) that can be changed during the execution of A program for addressing data from a program that can be in the same page as the instructions or in a different page. If an instruction has transferred to the delay register (35) the contents of the addressing registers (27 and 28) before intrusing a new addressing information into the address register (28) is followed by a Connection Instruction, the address of the following statement is given by the contents of the address register (28) and by the displacement address in this page given by the address field in the connection instruction. The instruction address register (4) content and the delay register (35) are transferred to the connecting register (6).
