<TITLE>
a resilient mounting for a reclining seat.
</TITLE>
<APPLICANT>

<ABSTRACT>
a resilient mounting for a reclining seat, the mounting  including a first member (10) to be secured to a supporting  base for the seat, a second member (29) to be secured to  the seat and which is pivotally connected to the first member,  and a torsion bar (17) which forms the pivot for the  two members and which provides a return bias when one  of the members is tilted relative to the other.  the torsion  bar (17) preferably comprises a metal bar which is bent to  form a first elongate portion (16) which provides the pivot  and a second elongate portion (30) which extends generally  alongside the first portion.
